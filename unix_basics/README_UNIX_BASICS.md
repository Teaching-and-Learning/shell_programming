# Unix Basics Practice

You can pratice the below commands using the Replit Shell or any shell in your local system. 

## Part A: Navigation

1. Run `pwd`.
2. Run `ls -lF`.
3. Go into the `data` folder.
4. Run `pwd` again.
5. Return to the parent folder.
6. Go to your home directory using `cd ~`.
7. Return to this repository folder.

Record the commands you used.

## Part B: Inspect files

1. Show the content of `documents/notes.txt`.
2. Show the first 5 lines of `logs_sample.txt`.
3. Count the lines and words in `documents/notes.txt`.

## Part C: Search

1. Find files under this folder that contain the keyword `UNIX`.
2. Find files under this folder that contain either `for`, `while`, or `if`.
3. Find all `.txt` files.

## Part D: Copy and move safely

Use only the `sandbox/` folder for this part. You can create a sandbox first under the unix_basics folder. 

1. Copy `data/search.txt` into `sandbox/`.
2. Rename the copy to `search_copy.txt`.
3. Remove `search_copy.txt` using `rm -i`.
